# 螺旋千斤顶自动化设计系统v1.0

#### 介绍
这是宁波大学机械设计课程设计的自动运算并输出结果的软件，目前只是能实现功能，UI和更多交互功能都还没做，我开发主要是拿来写我的期末大作业，开源希望能帮助到更多的小伙伴。

#### 软件架构
软件就只有一个,py文件，没啥技术含量。
